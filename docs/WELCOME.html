Configuración Supabase (Opcional)

Si quieres guardar el feedback en una base de datos:

  1. Crea una cuenta en Supabase
  2. Sigue las instrucciones en SUPABASE_SETUP.md
  3. Edita src/supabase.js con tus credenciales

Nota: El proyecto funciona perfectamente sin Supabase. El feedback simplemente no se guardará en una base de datos.

Publicar en GitHub Pages

  1. Sube el proyecto a GitHub
  2. Ve a Settings > Pages
  3. Selecciona rama main y carpeta /src
  4. ¡Listo! Tu sitio estará en línea
